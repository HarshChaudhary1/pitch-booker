# Pitch Booker

Smart Cricket Ground Booking & Performance Tracking Platform

## Features

- User Registration & Login
- Ground Booking
- Booking Management
- Performance Tracking
- Dynamic Leaderboard

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Project Overview

Pitch Booker is a web application that allows users to book cricket grounds, manage bookings, track player performance, and view rankings through a leaderboard system.
